Desarrollo web para móviles

Construyendo experiencias web móviles

Así que has oído que el mundo móvil es una especie de gran cosa, pero no estás seguro de cómo transformar tus aplicaciones web tradicionales diseñadas para escritorio en experiencias multi-dispositivo más eficaces y rápidas.

Este curso está diseñado para enseñar a los desarrolladores web lo que necesitan saber para crear experiencias web multi-plataforma

Está enfocado en construir aplicaciones web "mobile first" que correrán en diversas plataformas incluyendo:

  • iOS,
  • Android
  • y otros.

Registrarse para asistir


Qué aprenderé?

Después de completar esta clase construirás una aplicación web con una experiencia móvil de primera clase

Aprenderás lo que necesita para:

  • construir grandes experiencias web para dispositivos móviles,
  • usar las herramientas para probar el rendimiento,
  • aplicar tu conocimiento para tus nuevos proyectos en el futuro.
